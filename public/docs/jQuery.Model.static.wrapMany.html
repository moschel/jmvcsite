jQuery.Model.static.wrapMany  function     

 

Takes raw data from the server, and returns an array of model instances. Each item in the raw array becomes an instance of a model class.

$.Model.extend("Recipe",{
  helper : function(){
    return i*i;
  }
})

var recipes = Recipe.wrapMany([{id: 1},{id: 2}])
recipes[0].helper() //-> 1

If an array is not passed to wrapMany, it will look in the object's .data property.

For example:

var recipes = Recipe.wrapMany({data: [{id: 1},{id: 2}]})
recipes[0].helper() //-> 1
$.Model.wrapMany(instancesRawData) -> Array
{Array}

an array of raw name - value pairs.

{Array}

a JavaScript array of instances or a [jQuery.Model.List list] of instances if the model list plugin has been included.
