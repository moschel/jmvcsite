jQuery.fn.compare  function     

plugin: dom/compare
download: jquery.compare.js
test: qunit.html
 

Compares the position of two nodes and returns a bitmask detailing how they are positioned relative to each other. You can expect it to return the same results as compareDocumentPosition. Parts of this documentation and source come from John Resig.

A bitmap with the following digit values:

BitsNumberMeaning
0000000Elements are identical.
0000011The nodes are in different documents (or one is outside of a document).
0000102Node B precedes Node A.
0001004Node A precedes Node B.
0010008Node B contains Node A.
01000016Node A contains Node B.
